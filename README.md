A beautifully organized collection of C programming practice programs, notes, exercises, and assignments for beginners and students.

## ✨ About This Repository

This repository contains:

- Basic C programs and syntax practice
- Control statements, loops, arrays, and functions
- Pattern printing and logical programs
- Practice questions and assignments
- Notes and study materials for C programming

It is specially useful for BCA students and anyone starting their journey with C.

## 🚀 How to Use

1. Open any .c file in VS Code.
2. Compile it using GCC:

```bash
gcc filename.c -o output
./output
```

3. Explore the folders for practice programs, notes, and exercises.
