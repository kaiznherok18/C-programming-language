This repository is a beginner-friendly collection of C programming programs, notes, practice files, and assignments. It is designed especially for BCA students and anyone who wants to learn C step by step.

What is C Programming?

C is a powerful and popular programming language used to build: -

- Operating systems
- Compilers
- Embedded systems
- Game engines
- System software

It is known for being fast, simple, and close to the computer hardware.

📚 Step-by-Step Learning Path -

1. Learn the basic structure of a C program.
2. Understand variables and data types.
3. Practice input and output.
4. Learn operators.
5. Study conditional statements like if and else.
6. Practice loops such as for, while, and do-while.
7. Learn functions.
8. Work with arrays and strings.
9. Solve pattern programs and logic-based problems.

Your First C Program -

Create a file named `hello.c` and write this code:

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

Explanation -

- `#include <stdio.h>` includes the standard input/output library.
- `int main()` is the main function where program execution begins.
- `printf()` displays output on the screen.
- `return 0;` ends the program successfully.

▶️ How to Run a C Program in VS Code -

On Windows -

1. Open a terminal in VS Code.
2. Compile the file:

```bash
gcc hello.c -o hello.exe
```

3. Run the program:

```bash
hello.exe
```

On Window, Linux or Mac

```bash
gcc hello.c -o hello
./hello
```
